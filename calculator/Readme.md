# Assignment for Backend

Idea for this food calculation is to simulate how real practice ordering should work!


## Installation

Install project with npm/yarn/pnpm

```bash
  npm install
```

To get start

```bash
  npm run start:dev
```

To build and start

```bash
  npm run build
  npm run start
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
